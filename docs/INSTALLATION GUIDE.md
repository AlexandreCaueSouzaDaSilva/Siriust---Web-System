## GUIA DE INSTALAÇÃO


## 1. Requisitos Mínimos:
-   [PHP](https://www.php.net/downloads)
-   [Composer](https://getcomposer.org/download/) Gerenciador de dependências do PHP
-   Navegador Web atualizado (Chrome, Edge, Opera, Etc)

## 2. Passo a Passo Para Rodar o Projeto:
    Clonar Repositório:
    Digite --> git clone https://github.com/AlexandreCaueSouzaDaSilva/Siriust---Web-System.git

    Para entrar na pasta:
    Digite --> cd siriust

## 3. Estrutura geral do backend

/config - configu
/public
/src
    /Api
    /Application
    /Domain
Module.php


EXPLICAÇÃO SIMPLES:
src/
  onde fica o código do projeto

public/
  onde o navegador acessa (index.php)



## 4. Como rodar o servidor

Entra na pasta pública:

    cd public


Agora roda o servidor built-in do PHP:

    php -S localhost:8080
